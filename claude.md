# Adam Frank - Personal Portfolio

A modern personal website built with vanilla HTML, CSS, and JavaScript.

## Quick Start

1. Clone the repository:
```bash
git clone https://github.com/adamfrank321/afshome.git
cd afshome
```

2. Open `index.html` in your browser or start a local server:
```bash
python -m http.server 8000
# then visit http://localhost:8000
```

## Customization

### Update Your Info

Edit `index.html` to:
- Change name, title, and bio in the hero section
- Update skills in the about section
- Modify project descriptions and links
- Add your social media links

### Change Colors

Edit CSS variables in `styles.css`:
```css
:root {
    --accent: #0066cc;
    --text-primary: #1a1a1a;
    --bg-primary: #ffffff;
}
```

### Add Content

- **New writings**: Add to `pages/writings.html` articles list
- **New projects**: Add to `pages/projects.html` projects list
- **New sections**: Follow existing patterns in HTML structure
